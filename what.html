operating system (OS) is system software that controls computer hardware, software resources, also provides services for computer softwares.

Time-sharing operating systems schedule tasks for effective program use, which may include accounting software for processor time allocation, storage devices, printing, and other resources.

The operating system acts as an intermediary between programs and computer hardware for hardware functions such as I/O, and memory allocation, although the application code gets executed directly by the hardware yet often makes system calls to or is interrupted by an OS function.
